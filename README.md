# Hindsight

Check resources usage across multiple platforms.

## Running

### GCP

- Copy key file to `./gcloud.json`
- Set env variable before running the app
```shell
set GOOGLE_APPLICATION_CREDENTIALS=gcloud.json
```

### Application

Use
```shell
npm run start
```
