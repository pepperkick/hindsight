# Hindsight

Check resources usage across multiple platforms.

## Running

### Configure

```json5
{
  "lighthouseMongodbUri": "",  // URL to lighthouse mongodb
  "discordWebhook": ""         // Discord webhook to notify orphan resources
}
```

### Application

Use
```shell
npm run start
```
